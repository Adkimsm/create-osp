# Create-OSP

> Create (an) Open Source Project.

A Create Starter Template for Open Source Project, in NodeJS.

## Features:

- [x] Project Name
- [x] LICENSE of the Project
- [x] `README.md` (use `readme-md-generator`)
- [ ] `.gitignore` for the Project
